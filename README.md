## About Project


- myflickrapp.sqlite is the database and is on project root.
- When the page is opened, 10 images are loaded, these 10 images are automatically saved in the database without any user intervention.
- There is the possibility of searching images by size and saving them manually in the database.
- Do not forget that it may be necessary to edit the .env to change the database path. (DB_DATABASE=YourPath)
